# Expense & Income Tracker

A streamlined financial tracking tool that allows users to manage daily expenses and income with clarity and ease.

## 🌐 Live Demo
https://virginiamarc.com/p8/

## 🛠 Tech Stack
- HTML / CSS / JavaScript
- Local storage for saving entries
- Responsive layout

## ✨ Features
- Add income and expenses
- Categorize transactions
- View running totals
- Simple, user-friendly interface

## 📁 Project Purpose
This project highlights practical UI design and functional budgeting tools for everyday use.
